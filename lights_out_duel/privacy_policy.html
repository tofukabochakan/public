Privacy Policy

Last Updated: April 2026

This application “Lights Out Puzzle Duel” (the “App”) respects your privacy and handles user data in accordance with applicable laws and Google Play Developer Policies. This Privacy Policy explains what data the App collects, how it is used, how it is shared, and what choices users have.

1. Data We Collect

(1) Data voluntarily sent by the user

● When the Ranking Submission feature is ON

  • Nickname (entered by the user)
  • Game score
  • Hashed device identifier (irreversible hash)

When the feature is OFF, none of this data is sent.

● When the user submits a custom Lights Out puzzle

  • User-created Lights Out puzzle data
  • Hashed device identifier (irreversible hash)

● When the user “likes” or “unlikes” a puzzle created by another user

  • Like information
  • Hashed device identifier (irreversible hash)

(2) Data automatically collected by AdMob

  • Advertising ID (device ID)
  • Crash logs
  • Diagnostic information
  • Ad performance data

These data are sent to Google and used for advertising and analytics purposes.

2. Purpose of Data Use

(1) For App functionality

  • Displaying rankings
  • Publishing user-created puzzles
  • Providing the “like” feature
  • User identification and misuse prevention via hashed device identifiers

(2) Advertising and analytics (AdMob)

  • Displaying and optimizing ads
  • Improving App performance
  • Crash analysis

3. Data Sharing

● Data shared with third parties

  • Data collected by AdMob (Advertising ID, crash logs, diagnostic information, ad performance data) is shared with Google.

● Data NOT shared with third parties

  • Nickname
  • Score
  • User-created puzzles
  • Like information
  • Hashed device identifier

These data are used only within the App’s server.

4. User Choices

  • Users can turn the Ranking Submission feature ON or OFF. When OFF, nickname, score, and hashed device identifier are not sent.
  • If the user does not use the puzzle submission feature, puzzle data and hashed device identifier are not sent.
  • If the user does not use the “like” feature, like information iand hashed device identifier are not sent.
  • AdMob data collection cannot be disabled from within the App.

5. Data Security

  • All transmitted data uses encrypted communication (HTTPS).
  • Hashed identifiers use irreversible hashing and cannot be converted back to the original device ID.
  • Appropriate security measures are implemented to prevent unauthorized access and data leaks.

6. Data Deletion

Users may request deletion of their nickname, scores, puzzle data, or like information by contacting the email address below.

Email: tofukabochakan@gmail.com

7. Changes to This Policy

This Privacy Policy may be updated from time to time due to changes in laws or service content. When significant changes are made, we will notify users within the App or on the store page.

プライバシーポリシー

最終更新日：2026年4月

本アプリ「対戦 パズル ライツアウト」（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、 適用される法令および Google Play デベロッパーポリシーに従って、ユーザーのデータを適切に取り扱います。 本ポリシーでは、本アプリが収集するデータ、その利用目的、第三者との共有、ユーザーの選択肢について説明します。

1. 収集するデータ

（1）ユーザーが任意で送信するデータ

● ランキング送信機能を ON にした場合

  • ニックネーム（ユーザーが設定画面で入力したもの）
  • スコア（ゲームプレイ結果）
  • ハッシュ化されたデバイス識別子（不可逆ハッシュ）

ランキング送信機能を OFF にした場合、これらのデータは送信されません。

● ユーザーが作成したライツアウトの問題を投稿する場合

  • ユーザーが作成したライツアウトの問題データ
  • ハッシュ化されたデバイス識別子（不可逆ハッシュ）

● 他ユーザーの問題に「いいね」をつける場合、または、「いいね」を取り消す場合

  • 「いいね」情報
  • ハッシュ化されたデバイス識別子（不可逆ハッシュ）

（2）広告（AdMob）による自動収集データ

  • 広告ID（デバイスID）
  • クラッシュログ
  • 診断情報
  • 広告パフォーマンスデータ

これらのデータは Google に送信され、広告配信および分析のために使用されます。

2. データの利用目的

（1）アプリの機能のため

  • ランキング表示
  • ユーザー作成問題の公開
  • 「いいね」機能の提供
  • ハッシュ化IDによるユーザー識別および不正利用防止

（2）広告および分析（AdMob）

  • 広告の表示および最適化
  • アプリのパフォーマンス改善
  • クラッシュ解析

3. データの共有

● 第三者に共有されるデータ

  • AdMob が収集するデータ（広告ID、クラッシュログ、診断情報、広告パフォーマンスデータ）は Google に共有されます。

● 第三者に共有されないデータ

  • ニックネーム
  • スコア
  • ユーザー作成問題
  • 「いいね」情報
  • ハッシュ化されたデバイス識別子

これらは本アプリのサーバー内でのみ利用されます。

4. ユーザーの選択肢

  • ランキング送信機能を ON/OFF できます。OFF の場合、ニックネーム・スコア・ハッシュ化IDは送信されません。
  • 問題投稿機能を使用しない場合、問題データ・ハッシュ化IDは送信されません。
  • 「いいね」機能を使用しない場合、「いいね」情報・ハッシュ化IDは送信されません。
  • 広告（AdMob）によるデータ収集は、アプリ内からオフにすることはできません。

5. データの安全管理

  • 送信されるデータは HTTPS による暗号化通信を使用します。
  • ハッシュ化された識別子は不可逆ハッシュを用いており、元のデバイスIDに復元できません。
  • 不正アクセスや情報漏えいを防止するため、適切なセキュリティ対策を実施します。

6. データの削除

ユーザーは、ニックネーム・スコア・投稿データ・「いいね」情報などの削除を希望する場合、 下記の連絡先までお問い合わせいただくことで削除を依頼できます。

連絡先：tofukabochakan@gmail.com

7. プライバシーポリシーの変更

本ポリシーは、法令やサービス内容の変更等に応じて、予告なく改定される場合があります。 重要な変更がある場合は、本アプリ内またはストアページにてお知らせします。
